# 🐸 Orbion: Echoes of Chaos

Jogo de plataforma 2D desenvolvido em **Construct 2** para a disciplina de Desenvolvimento de Jogos.

## 📖 Sobre o Jogo
Orbion é um sapo verde que vive em um mundo corrompido por uma energia misteriosa.
Explore fases progressivas, supere obstáculos e descubra a origem do caos.

## 🎮 Mecânicas
- Movimentação com setas do teclado
- 5 vidas por fase
- 3 fases com dificuldade progressiva
- Coleta de itens e power-ups

## 🛠️ Tecnologia
- Engine: Construct 2
- Exportação: HTML5
- Plataforma: PC (navegador)
